Status

It’s been quite a while since I last checked up on my blog. I went online a while ago and realized the domain I was hosting my comment section on needs renewal. Some local source files are also somehow corrupted, and I can’t launch the blog locally.

Seeing how everything is falling apart, I might as well implement some big updates.

Changes

The biggest change is on the switch of theme, from Anatolo to ParticleX. I handwrote some more features, such as home page thumbnails, view counts, and last updated time. I considered using Flexblock along the way, but since a lot of the features I wanted has been added myself, I didn’t bother switching again.

I am currently using Vercel to deploy my comment section, meaning that it still functions. The only issue is that access within China is a lot harder, as Vercel is somehow blocked off there. This is also why I bought a custom domain for hosting in the first place.

Due to the big theme switch, some older posts would have formatting and tag updates as well.

All specific changes:

    • Random home page background image
      • Probably going to be more options in the future

      • Nice animations?
      • Scroll to top button
